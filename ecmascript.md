# Ecmascript

## Values Types

### Numbers
* -12 
* 0 
* 8

**Special numbers**
* Infinity
* -Infinity

### String 
* 'text'

**combine string** 
```javascript
`Tekst ${var}`
```

### Boolean
* true
* false

### Empty values
* null
* undefined
* NaN

### Object
json like format

```javascript
let list = {
  value: 1,
  rest: {
    value: 2,
    rest: {
      value: 3,
      rest: null
    }
  }
};
```


## Basic Math

* + add
* - minus
* * times
* / divide
* % modulo (remainder of a division)
* ++ adds one
* -- subtracts one

## Logical Operators

* && and
* || or
* == equal
* > greater than
* >= greater than or equal
* < smaller than
* <= smaller than or equal
* != not equal
* 
* === equal to (also data type)
* !== not equal to (also data type)
